# OpenCVAndroid

Use OpenCV in Android devices. This project packages the latest version up to this date (9 June 2019 19:35 GMT) which is OpenCV 4.1.0.

Just download and import the project in Android Studio then run it to generate the APK file. You can also download the APK file from [here](https://github.com/ahmedfgad/OpenCVAndroid/releases/download/1.0/OpenCVAndroid-CannyFilter.apk). 

After installing the app and run it in your device, the main window appears as below.

![Fig 28](https://user-images.githubusercontent.com/16560492/59163252-c43a6b80-8afe-11e9-86b6-31c50a073182.jpg)

Just click the button and the Canny filter will be applied. The result is as shown below.
